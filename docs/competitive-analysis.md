## 二、各产品核心卖点

### Magnet — 最简单易用

- 拖拽到屏幕边缘即刻吸附，零学习成本
- Mac App Store 独家，Apple 审核背书
- 支持 15+ 预设窗口位置
- 竖屏自动适配布局

### Rectangle Pro — 功能最丰富

- Window Throw：一个快捷键投掷到 16 种位置
- Snap Areas：屏幕任意位置创建吸附区
- Window Stashing：窗口隐藏到边缘，鼠标靠近滑出
- Workspace：一键排列多个应用 + 显示器连接自动触发
- iCloud 配置同步

### Moom 4 — 最灵活可定制

- Pop-up 面板可容纳 61 个动作（含嵌套文件夹）
- 模板系统：固定尺寸/位置，应用特定布局
- Hover 操作：按住修饰键移动/调整任意后台窗口
- 可禁用系统全屏按钮

## 三、MyScreen 的差异化优势

MyScreen 与上述三者**不是直接竞争**，而是解决不同问题：

| 它们解决的问题 | MyScreen 解决的问题 |
|--------------|-------------------|
| 快速排列窗口到各种位置 | **永久保留屏幕区域给特定应用** |
| 用完即走，不持续管理 | **持续保护，其他窗口无法侵入** |
| 手动操作为主 | **自动化：绑定应用自动归位** |

### MyScreen 的独有价值（竞品都没有的）

1. **屏幕保留区** — 划出永久区域，不被任何窗口占用
2. **应用绑定** — 指定应用自动锁定在保留区
3. **窗口约束** — 其他窗口自动避开保留区
4. **拖拽分割线** — 实时调整分区比例
5. **一键隐藏/显示** — 快捷键切换全屏工作模式
